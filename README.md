# Fantera Collection 

This is test contracts for Fantera

## Before starting
Create `.env` file and fill the values from `.env.example`.

## Deployment & Testing
### Command lines for deployment
```
yarn
yarn compile
yarn deploy-goerli
```

### Test contracts

```
yarn test
```

### Mint ERC20 token

```
npx ts-node scripts/mintERC20.ts
```
### Mint ERC721 token

```
npx ts-node scripts/mintERC721.ts
```
